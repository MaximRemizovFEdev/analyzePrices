# analyzePrices
Аналзатор цен по ссылкам

# Общая информация
- Приложение: веб‑сервис для сравнения цен по введённым ссылкам.
- Цель: дать пользователю быстрый способ собрать в одном месте цену товара с разных сайтов, сохранить историю и анализировать изменения цен.

# Основной сценарий использования
- Пользователь открывает приложение.

- Вводит название/обозначение предмета в поле «Ищем».

- Добавляет одну или несколько ссылок на страницы, где есть информация об этом предмете и его цене.

- Нажимает кнопку «Сравнить».

# Приложение:

- Для каждой ссылки получает текстовое содержимое страницы (через бэкенд/прокси).

- Вызывает внешнее AI/LLM API для извлечения из текста названия товара и цены.

- Сохраняет результат (название, цена, ссылка, дата анализа) в localStorage.

- Пользователь видит таблицу сравнения цен и базовую аналитику (графики/статистика).

- В будущем пользователь возвращается, выбирает ранее сохранённые наборы ссылок и смотрит, как цены менялись с течением времени.
